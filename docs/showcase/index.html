About PaperBanana

An enhanced multi-agent pipeline for publication-quality academic diagrams.

What it is: A Python tool that generates diagrams from text descriptions. You describe a concept, it produces a publication-quality image.

What makes it different: The Planner agent asks "What is this concept LIKE?" before drawing. It finds a visual metaphor (security = water purification, containers = shipping crates) that makes the concept click instantly.

Original framework: Peking University + Google Cloud AI Research

Storytelling enhancement: Community contribution by @stuinfla

Models used: Google Gemini 3 Pro (reasoning) + Gemini 3 Pro Image (generation)

GitHub: github.com/stuinfla/paperbanana
